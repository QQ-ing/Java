# 01.ClassWithOnlyPrivateConstructorsShouldBeFinal

如果Class只包含私有的构造方法，则Class应该使用final进行修饰，除非构造方法在内部类中被调用了。

例如：

```Java
public class Foo {
    private Foo(){
        // init
    }
}
```

建议改为：

```Java
public final class Foo {
    private Foo(){
        // init
    }
}
```
