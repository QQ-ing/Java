# 02.DoNotConcatStringInLog

不要在日志中拼接字符串，例如：

```Java
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

public class Foo {
	private static final Logger logger = LoggerFactory.getLogger(Foo.class);

	public void method(int num) {
		logger.debug("num is " + num);
	}
}
```

应该改成使用占位符代替字符串拼接：

```Java
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

public class Foo {
	private static final Logger logger = LoggerFactory.getLogger(Foo.class);

	public void method(int num) {
		logger.debug("num is {}", num);
	}
}
```
