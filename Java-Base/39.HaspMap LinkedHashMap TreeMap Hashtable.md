# 39.HaspMap LinkedHashMap TreeMap Hashtable

* HaspMap

```
HashMap是最常用的Map，底层使用数组+链表实现。它根据键的hashCode值存储数据，具有数组的随机存取特征。
+哈希碰撞
+碰撞后处理
+装载因子
+扩容rehash
```

* LinkedHashMap

```
LinkedHashMap会保存插入顺序，使用iterator遍历时会根据插入顺序输出。
```

* TreeMap

```
红黑树实现的Map，可以根据比较器对TreeMap进行排序。
```

* Hashtable

```
同步版本的HashpMap
```
